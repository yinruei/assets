# 專案名稱：全方位財務目標規劃機

## 專案說明
這個專案是一個財務目標規劃機，旨在幫助用戶計算和規劃他們的財務目標。用戶可以輸入資產總目標、初始投資金額、每月定期定額、年化報酬率以及目前年齡，系統將根據不同的計算模式提供相應的結果。

## 檔案結構
- `index.html`：網站的主要入口檔案，包含網站的結構和內容，並引入 CSS 和 JavaScript 檔案。
- `css/styles.css`：網站的樣式設定，定義各種元素的外觀和佈局。
- `js/main.js`：網站的 JavaScript 代碼，負責處理用戶互動和動態功能。
- `assets`：存放圖片、圖標或其他媒體資源的資料夾。
- `.github/workflows/deploy.yml`：GitHub Actions 的工作流程檔案，用於自動化部署過程。
- `README.md`：專案的說明和使用指南。
- `LICENSE`：專案的授權條款，說明使用該專案的條件和限制。

## 使用指南
1. **克隆專案**：
   使用以下命令將專案克隆到本地：
   ```
   git clone https://github.com/yinruei/assets.git
   ```

2. **安裝依賴**：
   根據需要安裝任何必要的依賴。

3. **啟動網站**：
   打開 `index.html` 檔案於瀏覽器中查看網站。
